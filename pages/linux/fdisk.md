# fdisk
> Manipulate disk partition table.

- Example description:

`fdisk -l /dev/sda`

- List  the  partition  tables for the sda disk and then exit:

`fdisk -s /dev/sda1`

- Print the size (in blocks) of /dev/sda1:

`fdisk /dev/sda`
- Create/remove/change partitions in the device:    
      
`fdisk -h`
- Print help and then exit.
